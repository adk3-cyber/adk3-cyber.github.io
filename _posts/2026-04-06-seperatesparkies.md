**Assignment overview**

This assginment is preparin us to be able to make one big sparkie project with the three patterns of firely lights going simultaneously on the arduino. For this project we simply had to simulate each one by one using the arduino. In order to do this we had to use state based coding which allows the computer to tell wheather or not the light is on or flashing etc. 

**Tip or trick**
Double check that you don't mispel variable names and don't make variable names too similiar to one another.
