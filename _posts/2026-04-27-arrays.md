**Assignment overview**
This assignemnt requires us to write code for our arduino which turns on a random PWM pin every second with a random brightness. If the pin is already on, the program turns it off. 

**Tip or trick**
Make sure that there's an equal number of starting values at 0 for your second array as there are total pins.
